# 倪柝声文集 · 阅读器 — 产品设计文档

## 产品概述

一款专注于倪柝声文集阅读的跨平台 App，收录全部 3 系列 · 62 册 · 1,325 章著作。提供沉浸式阅读体验、全局搜索、阅读统计和历史记录。

**平台**: H5 / Android / iOS  
**技术**: React + Capacitor 跨平台方案

---

## 用户画像

- 对倪柝声著作有深度阅读需求的基督徒
- 需要全文搜索快速定位经文引用和主题
- 希望追踪阅读进度，长期持续阅读
- 偏好简洁、无干扰的阅读界面

---

## 信息架构

```
首页（文集）
├── 继续阅读卡片
├── 第一辑 · 早期著作（20 册）
├── 第二辑 · 中期著作（26 册）
└── 第三辑 · 晚期著作（16 册）
     └── 书卷列表
          └── 章节目录
               └── 阅读器

搜索
├── 搜索首页（L1）— 全部结果按书分组
├── 书卷筛选（L2）— 单书匹配章节
└── 文本定位（L3）— 关键词高亮 + 下一处跳转

我的
├── 阅读统计（4 卡片）
├── 阅读历史（按日分组）
└── 阅读设置（主题 / 字号）
```

## 页面设计

### 首页（文集）

- **继续阅读卡片**: 绿色背景，显示最近阅读的章节名和所属书名，点击直接续读
- **系列卡片**: 左侧绿色竖线装饰，显示系列名、时期、册数和总章节数
- **导航**: 底部 Tab 栏

### 书卷列表

- 某系列下所有书的列表
- 每项显示书名和章节数
- 点击进入章节目录

### 章节目录

- 某书下所有章节列表
- 已读章节前显示 ● 标记
- 点击进入阅读器

### 阅读器

- **全屏沉浸**: 隐藏底部 Tab 栏
- **控制栏**: 点击屏幕中间区域切换显示/隐藏
  - 顶部栏: 返回按钮 + 书名 + 章节进度（如 3/20）
  - 底部栏: 主题切换（白/护/暗）+ 字号切换（小/中/大/特大）+ 目录按钮
- **底部进度条**: 阅读进度百分比
- **右侧滚动指示器**: 滚动时显示，1.2 秒后自动淡出
- **章节导航**: 文末上一章/下一章按钮
- **阅读计时**: 页面可见时自动计时，后台暂停

### 搜索

- **搜索栏**: 左侧书卷选择器 📂 + 搜索框 + 清除按钮
- **L1 搜索结果**: 按系列 → 书 层级分组，每书最多展示 5 条匹配章节，点击「查看全部」进入 L2
- **L2 书卷筛选**: 仅显示某本书的匹配章节，支持返回 L1
- **L3 文本定位**: 章节全文展示，关键词高亮，顶部显示「当前匹配/总匹配」和下一处按钮，右侧浮动快速跳转按钮
- **书卷选择器**: 底部抽屉面板，系列 → 书 二级选择
- **系统返回键**: L3 → L2/L1 → 退出搜索，完整拦截

### 我的

- **阅读统计**: 4 格卡片（累计阅读 / 已读章节 / 连续天数 / 今日阅读）
- **阅读历史**: 最近 50 条按日期分组，默认折叠，点击查看全部
- **阅读设置**: 主题三选一 + 字号四选一

## 数据与离线

- 全部 62 本书内容预置在安装包内，首次安装即可离线使用
- 阅读进度、统计、历史存储在 localStorage，跨会话持久化
- 搜索索引预构建，首次打开搜索时懒加载
